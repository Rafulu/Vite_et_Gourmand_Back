# ECF 2026 - Back

## Prérequis
- Docker Desktop installé

## Lancer le projet en local
1. Cloner le repo
2. Se placer à la racine du projet `ecf-2026`
3. Lancer la commande `docker-compose up -d --build`

## Accès
- Back PHP : http://localhost:8000
- PhpMyAdmin : http://localhost:8080
- MongoDB : port 27017
